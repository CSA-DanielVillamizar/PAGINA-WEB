# Monorepo Fundación L.A.M.A. Medellín

Este monorepo contiene una implementación base con:

- Frontend: Next.js 14 (App Router) + TypeScript + Tailwind CSS + shadcn/ui + React Hook Form + Zod + Zustand
- Backend: Node.js + Express + TypeScript + Prisma + PostgreSQL + JWT (access/refresh)

Estructura general:

- frontend/: Aplicación Next.js
- backend/: API Express + Prisma

Instrucciones de arranque están en los README de cada paquete.
